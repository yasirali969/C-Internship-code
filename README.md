# Menu Driven 
Title: Menu-Driven C++ Program
**Objective**:
The objective of this project is to develop a menu-driven program in C++ that performs multiple basic operations such as a calculator,
even/odd number checking, and a grading system.
**Description**:
This program is designed using a menu-driven approach where the user is presented with multiple options. Based on the user's choice, the
program performs different tasks. The program uses fundamental programming concepts such as conditional statements, loops, and switch-case structures.
Features:
1.	Calculator:
The calculator performs basic arithmetic operations including addition, subtraction, multiplication, and division. It takes two integer inputs and an operator from the user and displays the result accordingly.
2.	Even/Odd Checker:
This feature allows the user to input a number and determines whether the number is even or odd using the modulus operator.
 
3.	Grading System:
The grading system calculates the grade of a student based on the marks entered by the user. Grades are assigned according to
predefined conditions.
4.	Exit Option:
The program provides an option to exit the system.
**Tools & Technologies Used:**
•	Programming Language: C++
•	IDE: VS Code /
